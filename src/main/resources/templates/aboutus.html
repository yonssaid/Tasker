About Us

Hey there! Welcome to Tasker

Tasker is a project born out of my own experience as a busy college student trying to juggle classes, assignments, and personal projects. I wanted a simple, effective tool to help me stay organized and keep track of everything I needed to do. So, I decided to create Tasker, a task management app that’s all about making life a little bit easier.

The Journey

Building Tasker has been a mix of challenges and victories. Here’s how it all went down:

  • Initial Setup: I kicked things off with Spring Boot for the backend and React for the frontend, aiming for a solid foundation.
  • Database Transition: Switching from MySQL to PostgreSQL was a big learning curve, but it was worth it for the advanced features and stability.
  • Security: Implementing JWT for user authentication was tough but crucial for making sure Tasker is secure.
  • UI Design: Crafting an interface that’s both functional and pleasant to use has been an ongoing process, with lots of tweaks and improvements along the way.

What’s Next?

I’m always thinking about ways to improve Tasker. Some ideas for the future include:

  • Collaboration Features: Letting users work together on tasks.
  • Better Insights: Adding features to help you track your productivity.
